# Sprint 11

Goal

Implement full Dark Mode, Framer Motion page transitions, zero-dependency sound effects, and an enhanced off-screen Share Card format to round out Phase 0's visual polish and game feel.

---

Epic

Polish, Visuals, and Feel

---

Task 1

Wire up theme detection/toggling to the HTML container in `AppProvider` and support Tailwind dark mode on all screens.

---

Task 2

Add page transitions using Framer Motion animations.

---

Task 3

Create a client-side synthetic `useSound` hook utilizing `AudioContext` to trigger button click, timer tick, and result success sounds when sounds are enabled.

---

Task 4

Upgrade the Share Card system to construct a hidden, highly styled off-screen element for high-quality PNG generation.

---

Definition of Done

Dark/light/system theme choices are fully functional across the app.

Animations and screen transitions are smooth.

Dynamic sounds trigger on interaction without external MP3 asset dependency.

Generated share PNG is well-aligned and contains rich branding.
